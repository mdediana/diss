Arquivo LaTeX de exemplo de dissertação/tese a ser apresentados à CPG do IME-USP

Arquivo principal: 
  -'tese-exemplo.tex'

Arquivos dos capítulos e apêndice:
  - 'cap-introducao.tex'
  - 'cap-conceitos.tex'
  - 'cap-conclusoes.tex'
  - 'ape-conjuntos.tex'

Arquivo de bibliografía: 
  -'bibliografia.bib'

Diretório de figuras: 
  - './figuras/'
  
Compilação do documento:
  - make pdf : usando pdflatex
  - make ps  : usando latex
  Ex:
      $ make pdf
  Obs: Para a compilação do documento que referencia figuras em formato 'ps' ou 'eps' deve-se usar: 'make ps'
